The whole package includes:
(1)Two source codes named “Transportation Hub.cpp” and “Data generator.cpp” in the “Code” folder.
(2)A report “Project3 report.pdf” in the “Documents” folder. 

Instructions:
(1)Open the report “Project3 report.pdf” and read.
(2)Open “Data generator.cpp” in your IDE.Change the const N(N cities),T(T queries) and K(threshold of transportation hub) , compile and generate the data in the file “test.in”.
(3)Open “Transportation Hub.cpp” in your IDE , compile and run.
